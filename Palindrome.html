Palindrome Checker

The Palindrome Checker is a web application designed to determine whether a given string is a palindrome. The app allows users to input a string and check if it reads the same forwards and backwards, ignoring spaces, punctuation, and capitalization.

Key Features:

  • Checks if a string is a palindrome.
  • Handles spaces, punctuation, and capitalization.
  • User-friendly interface for easy input and results.
  • Responsive design for optimal use on both desktop and mobile devices.

The app uses JavaScript to perform palindrome checks and handle string manipulations. CSS is used to style the user interface, providing a clean and modern look. The design is mobile-first, ensuring compatibility across a variety of devices.
